  • callback
)
async

Defined in lib/agent.js:40

Find agents filtered by the passed query and limited to and sorted by the passed options.

Parameters:

  • session Object

    An open session with a Nitrogen service.

  • query Object

    A query using MongoDB query format.

  • options Object

    Options for the query.

    • limit Number

      The maximum number to be returned.

    • sort String

      The field that the results should be sorted on.

    • dir Number

      The direction that the results should be sorted.

    • skip Object

      The number of results that should be skipped before pulling results.

  • callback Object

    Callback at completion of execution.

    • err Object

      If the find failed, find will callback with the error.

    • agents Array

      The set found with this query.
