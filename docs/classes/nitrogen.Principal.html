nitrogen.Principal Class

Defined in: lib/principal.js:4

A Principal in Nitrogen represents entities with authentication privileges in Nitrogen. Applications, devices, and services are all examples of principals.

  • callback
)
async

Defined in lib/principal.js:18

Authenticate this principal with the service. The mechanism used to authenticate depends on the type of principal. For users, an email and password is used, otherwise the secret generated during creation is used.

Parameters:

  • config Object

    The config data for the Nitrogen service.

  • callback Object

    Callback for the authentication.

    • err Object

      If the authentication failed, this will contain the error.

    • principal Object

      This principal.

    • accessToken Object

      The current access token to use with the Nitrogen service for this principal.

find

  • session
  • query
  • options
  • callback
)
async

Defined in lib/principal.js:82

Find principals filtered by the passed query and limited to and sorted by the passed options.

Parameters:

  • session Object

    An open session with a Nitrogen service.

  • query Object

    A query filter for the principals defined using MongoDB query format.

  • options Object

    Options for the query.

    • limit Number

      The maximum number of principals to be returned.

    • sort String

      The field that the results should be sorted on.

    • dir Number

      The direction that the results should be sorted.

    • skip Object

      The number of results that should be skipped before pulling results.

  • callback Object

    Callback at completion of execution.

    • err Object

      If the find failed, find will callback with the error.

    • principals Array

      The set of principals found with this query.
